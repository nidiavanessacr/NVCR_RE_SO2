# Blockchain como SOD

Es un registro confiable entre entidades autorizadas
# Testnet
Red de prueba de una blockchain donde se pueden experimentar funciones sin usar criptomonedas reales. Ideal para desarrolladores que quieren probar contratos inteligentes o transacciones sin riesgo
# Cadena de bloques o blockchain
Es una base de datos distribuida y segura que registra transacciones en bloques enlazados cronológicamente. Cada bloque contiene un conjunto de transacciones verificadas
# Hash criptográfico
Función matemática que convierte datos en una cadena alfanumérica única. Se usa para asegurar la integridad de la información en blockchain.
# Minería
Proceso mediante el cual se validan transacciones y se agregan nuevos bloques a la blockchain. Los mineros compiten para resolver problemas criptográficos y reciben recompensas.
# Consenso
Mecanismo que permite a los nodos de una blockchain acordar el estado actual de la red. Ejemplos: Proof of Work (PoW), Proof of Stake (PoS), Delegated Proof of Stake (DPoS).
# Nodos
Computadoras conectadas a la red blockchain que almacenan una copia del libro mayor y participan en la validación de transacciones.
# Transacción
Movimiento de valor o información dentro de la blockchain. Puede ser el envío de criptomonedas o la ejecución de un contrato inteligente.
# Firma digital
Método criptográfico que garantiza que una transacción fue creada por el propietario de una clave privada. Asegura autenticidad y no repudio.
# Smart contract (contratos)
Programa que se ejecuta automáticamente en la blockchain cuando se cumplen ciertas condiciones. Permite automatizar procesos sin intermediarios.
# Token
Representación digital de un activo en una blockchain. Puede ser una criptomoneda, un NFT, o un token de utilidad o gobernanza.
# Criptomoneda
Moneda digital que utiliza criptografía para asegurar transacciones y controlar la creación de nuevas unidades. Ejemplo: Bitcoin, Ethereum.
# Plataformas de blockchain
Redes que permiten la creación y ejecución de aplicaciones descentralizadas (dApps).

- Ethereum: Plataforma pionera en contratos inteligentes. Usa PoS y tiene una gran comunidad de desarrolladores.
- Solana: Blockchain rápida y de bajo costo, ideal para aplicaciones de alto rendimiento.
- Cardano: Plataforma basada en investigación académica, con enfoque en sostenibilidad y escalabilidad.
- NEAR: Blockchain centrada en la facilidad de uso y desarrollo, con contratos inteligentes en WebAssembly.
# Gobernanza
Sistema mediante el cual se toman decisiones sobre el desarrollo y evolución de una blockchain. Puede ser on-chain (votación con tokens) o off-chain (comités).
# Gas
Unidad que mide el costo computacional de ejecutar operaciones en Ethereum. Los usuarios pagan gas para realizar transacciones o ejecutar contratos.
# Exploradores de bloques
Herramientas web que permiten visualizar transacciones, bloques, contratos y direcciones en una blockchain. Ejemplos: Etherscan, Solscan, Cardanoscan.
# Hiperledger
Proyecto de código abierto respaldado por la Linux Foundation que ofrece frameworks para blockchains empresariales. No es una blockchain pública como Ethereum.
# Algorand
Blockchain que utiliza un mecanismo de consenso llamado Pure Proof of Stake. Es rápida, segura y adecuada para aplicaciones financieras y empresariales.
