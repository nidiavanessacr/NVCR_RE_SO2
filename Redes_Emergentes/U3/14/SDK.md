Kit de desarrollo de software
Un SDK (Software Development Kit) es un ==conjunto de herramientas de desarrollo de software que permite a los programadores crear aplicaciones para una plataforma específica, como un sistema operativo (iOS, Android) o un lenguaje de programación==

### Testnet
Es una red de prueba o una versión de prueba de una blockchain que no usa activos reales y permite a los desarrolladores probar sus aplicaciones y el protocolo en un entorno seguro y controlado.
### Mainet
Es una red blockchain completamente operativa e independiente donde se llevan a cabo transacciones reales con criptomonedas, contratos inteligentes y aplicaciones descentralizadas (dApps).
### Diferencia
**Testnet** es una versión de prueba de una red blockchain, utilizada por desarrolladores para probar cambios sin riesgo, mientras que **mainnet** es la red pública y real donde ocurren transacciones con valor monetario.
