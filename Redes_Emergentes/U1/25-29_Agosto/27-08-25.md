## ¿Qué es?

**TCP-IP:** Base de internet
**UDP:** Comunicación rápida, sin garantía de entrega.
**HTTP / HTTPS:** Navegación web
**FTP / SFTP:** Transferencia de archivos
**SMTP / IMAP / POP3:** Correo electrónico
**WiFi:** Tecnología de redes inalámbricas que utiliza ondas de radio para conectar dispositivos a una red local o a internet, sin necesidad de cables físicos
**Bluetooth:** Tecnología inalámbrica que permite conectar y comunicarse entre dispositivos electrónicos a corta distancia utilizando ondas de radio de 2.4 GHz.
**Zigbee:** Protocolo de comunicación inalámbrica para dispositivos IoT y domótica que funciona en redes de malla de baja potencia y alto rendimiento, basada en el estándar IEEE 802.15.4.
**LoRaWAN:** Protocolo de comunicación de bajo consumo energético y largo alcance (LPWAN) que se utiliza para conectar dispositivos del Internet de las Cosas (IoT) en aplicaciones como Smart Cities, agricultura y logística.
**NFC:** Tecnología de comunicación inalámbrica de corto alcance que permite la transferencia de datos entre dos dispositivos simplemente acercándolos.
**MQTT:**  Protocolo de mensajería ligero de publicación/suscripción para el Internet de las Cosas (IoT) y dispositivos con recursos limitados.
**SSL:** Protocolo de seguridad que cifra las comunicaciones entre un navegador y un servidor web, protegiendo la privacidad y la integridad de los datos transmitidos.
**Ethereum:** Plataforma de código abierto basada en cadena de bloques que permite la creación y ejecución de contratos inteligentes y aplicaciones descentralizadas.​
**WPA2:** Protocolo de seguridad para redes inalámbricas (Wi-Fi) que utiliza el estándar AES para cifrar datos, proporcionando una mayor protección de la información que sus predecesores WEP y WPA.
