# Características de la virtualización

- **Portabilidad:** La capacidad de un software de ser ejecutado en diferentes máquinas a sistemas operativos.
- **Escalabilidad:** Es la capacidad de un sistema, proceso o tecnología de adaptarse y crecer al aumentar la carga de trabajo o la demanda, manteniendo un buen rendimiento sin perder eficiencia.
- **Flexibilidad de entornos:** Capacidad para adaptarse a los requisitos de hardware.
- **Snapshots y clonación:** Es una copia instantánea del estado de un sistema, son archivos de solo lectura que se crean rápido y de forma eficiente, útiles para la protección de datos, restaurar un sistema y para pruebas de software.

# Características de los sistemas operativos distribuidos
- **Transparencia:** La característica que oculta al usuario y al programador la naturaleza compleja de un servidor distribuido
- **Recursos compartidos:** Los componentes se comunican a través de una red y comparten recursos como hardware, software o datos.
- **Escalabilidad:** La capacidad del sistema puede aumentarse fácilmente agregando nuevos nodos o servidores, lo que lo hace adecuado para cargas de trabajo cambiantes.
